Коротка інформація про ОС GNU/Linux

GNU/Linux — це вільна Unix-подібна операційна система, що складається з ядра Linux (створене Лінусом Торвальдсом у 1991 році) та програмного забезпечення проєкту GNU (який почався у 1984 році під егідою Фонду Вільного Програмного Забезпечення).

Перелік найбільш популярних дистрибутивів

Перелік найбільш популярних дистрибутивів

  • • Arch Linux – максимально кастомізований, мінімальний набір програм, AUR, не для новачків.
  • • Debian – один із найстаріших і найстабільніших, тільки вільне ПЗ, бездоганний для програмістів, багато посібників.
  • • Raspbian (Raspberry Pi OS) – дефолт для Raspberry Pi, підходить для навчання (Scratch, Python, Minecraft Pi).
  • • Gentoo – мета-дистрибутив, компілюється з вихідного коду, повна оптимізація під залізо, гнучкість.
  • • Ubuntu – ідеальний для новачків, зручний інтерфейс, Snap-пакети, портал для розробників.
  • • Fedora – похідний від Red Hat, має Developer Portal, вбудований Eclipse IDE, зручний для різних мов програмування.
  • • OpenSUSE – для розробників та адміністраторів, YaST, OBS, встановлені редактори (Vim, Emacs) та інструменти.
  • • CentOS – безкоштовний аналог RHEL, дуже стабільна, Developer Toolset, підтримка XEN-віртуалізації.
  • • Solus – rolling release з контрольованими оновленнями, підтримує IDE (Atom, Idea), багато мов програмування.
  • • Puppy Linux – надлегкий (≈330 МБ), працює в RAM, мінімальні пакети + можливість додавати devx, підтримує BASIC → C (BaCon), Bash, Python.

Приклади команд роботи в Linux CLI

  • Основні команди
  • • pwd – показує поточний робочий каталог
  • echo – виводить текст у термінал
  • ls – показує список файлів і папок
  • cd – змінює поточний каталог
  • touch – створює новий порожній файл
  • mkdir – створює нову директорію
  • cp – копіює файли та папки
  • mv – переносить або перейменовує файли
  • rm – видаляє файли та папки
  • history – показує історію виконаних команд

BASH-cкрипт, що імітує авторизацію в системі в текстовому режимі

  1. #!/bin/bash
  2. # Імітація авторизації з 3 спробами
  3. MAX_TRIES=3
  4. TRIES=0
  5. SUCCESS=0
  6. echo "Ласкаво просимо! Будь ласка, авторизуйтесь."
  7. while [[ $TRIES -lt $MAX_TRIES ]];
  8. do
  9. read -p "1) Логін: " login
  10. read -sp "2) Пароль: " pass
  11. echo ""
  12. if [[ "$login" == "admin" && "$pass" == "1234" ]]; then
  13. echo "Авторизація успішна! Вітаємо, $login."
  14. SUCCESS=1
  15. break
  16. else
  17. TRIES=$((TRIES + 1))
  18. echo "Невірний логін або пароль. Спроба $TRIES з $MAX_TRIES."
  19. fi
  20. done
  21. if [[ $SUCCESS -eq 0 ]]; then
  22. echo "Вхід заблоковано. Спробуйте пізніше."
  23. fi

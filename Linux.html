Коротка інформація про ОС GNU/Linux

GNU/Linux — це вільна Unix-подібна операційна система, що складається з ядра Linux (створене Лінусом Торвальдсом у 1991 році) та програмного забезпечення проєкту GNU (який почався у 1984 році під егідою Фонду Вільного Програмного Забезпечення). Основні характеристики: • Багатозадачність і багатокористувацькість (одночасно може працювати багато програм і користувачів). • Надійність та безпека (менш схильна до збоїв і атак, ніж багато інших ОС). • Відкритий код і свобода вибору (користувач може змінювати, поширювати й підбирати програми, робочі середовища, інтерфейси). • Належить спільноті – розробляється добровольцями та організаціями у всьому світі, а не однією компанією. Сьогодні GNU/Linux використовується на серверах, у бізнесі та дедалі частіше — вдома.

Перелік найбільш популярних дистрибутивів

Перелік найбільш популярних дистрибутивів

  • Популярні дистрибутиви GNU/Linux • Arch Linux – максимально кастомізований, мінімальний набір програм, AUR, не для новачків. • Debian – один із найстаріших і найстабільніших, тільки вільне ПЗ, бездоганний для програмістів, багато посібників. • Raspbian (Raspberry Pi OS) – дефолт для Raspberry Pi, підходить для навчання (Scratch, Python, Minecraft Pi). • Gentoo – мета-дистрибутив, компілюється з вихідного коду, повна оптимізація під залізо, гнучкість. • Ubuntu – ідеальний для новачків, зручний інтерфейс, Snap-пакети, портал для розробників. • Fedora – похідний від Red Hat, має Developer Portal, вбудований Eclipse IDE, зручний для різних мов програмування. • OpenSUSE – для розробників та адміністраторів, YaST, OBS, встановлені редактори (Vim, Emacs) та інструменти. • CentOS – безкоштовний аналог RHEL, дуже стабільна, Developer Toolset, підтримка XEN-віртуалізації. • Solus – rolling release з контрольованими оновленнями, підтримує IDE (Atom, Idea), багато мов програмування. • Puppy Linux – надлегкий (≈330 МБ), працює в RAM, мінімальні пакети + можливість додавати devx, підтримує BASIC → C (BaCon), Bash, Python.

Приклади команд роботи в Linux CLI

  • Основні команди • pwd – показує поточний робочий каталог pwd # /home/user/Documents echo – виводить текст у термінал echo "Hello, Linux!" # Hello, Linux ls – показує список файлів і папок cd – змінює поточний каталог touch – створює новий порожній файл mkdir – створює нову директорію cp – копіює файли та папки mv – переносить або перейменовує файли rm – видаляє файли та папки history – показує історію виконаних команд

10 базових команд роботи в Git

  1. git init – створює новий локальний репозиторій.
  2. git clone – завантажує копію віддаленого репозиторію.
  3. git branch – робота з гілками.
  4. git checkout – перехід між гілками.
  5. git add – додає зміни у staging area.
  6. git commit – зберігає зміни у локальному репозиторії.
  7. git status – показує стан робочої директорії.
  8. git log – історія комітів.
  9. git pull – отримання оновлень з віддаленого репозиторію.
  10. git push – надсилання змін у віддалений репозиторій.
